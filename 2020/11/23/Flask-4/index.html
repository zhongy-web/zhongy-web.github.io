数据库操作

创建表

db.create_all() #寻找所有db.Model子类，并在数据库中创建对应的表，如果已经存在，使用后不会更新或者重新创建，需要删除后再创建才能更新。

db.drop_all() #删除所有表

插入行

admin_role = Role(name='Admin') # 添加到会话当中

db.session.add(admin_role) # 这个就是事务

db.session.commit() # 提交到数据库中

db.session.rollback() # 数据回滚

修改行

admin_role.name = ‘Administrator’

db.session.add(admin_role)

db.session.commit()

删除行

db.session.delete(mod_role)

db.session.commit()

查询行

Role.query.all() # 查询所有

User.query.filter_by(role=user_role).all() # 使用过滤器

john = User.query.filter_by(username='john').first() #将一条查询结果封装成python对象

常用的SQLAlchemy查询过滤器

过滤器 说明
filter() 把过滤器添加到原查询上，返回一个新查询
filter_by() 把等值过滤器添加到原查询上，返回一个新查询
limit 使用指定的值限定原查询返回的结果
offset() 偏移原查询返回的结果，返回一个新查询
order_by() 根据指定条件对原查询结果进行排序，返回一个新查询
group_by() 根据指定条件对原查询结果进行分组，返回一个新查询

常用的SQLAlchemy查询执行器

方法 说明
all() 以列表形式返回查询的所有结果
first() 返回查询的第一个结果，如果未查到，返回None
first_or_404() 返回查询的第一个结果，如果未查到，返回404
get() 返回指定主键对应的行，如不存在，返回None
get_or_404() 返回指定主键对应的行，如不存在，返回404
count() 返回查询结果的数量
paginate() 返回一个Paginate对象，它包含指定范围内的结果
